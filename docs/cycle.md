# 홈트레이닝 사이클

사이클과 BLE 통신으로 운동 데이터를 측정하는 앱 | 2019.06

## 프로젝트 개요

홈트레이닝 사이클 기기와 BLE로 통신하며 운동 데이터를 수집하고 관리하는 Android 앱을 개발했습니다.

단말 통신, 로컬 저장, 원격 동기화가 연결된 구조를 구현하며 하드웨어 연동형 앱을 개발했습니다.

## 🛠 기술 스택

| 항목 | 기여도 |
|------|--------|
| Java / Android | 100% |
| SQLite | 100% |
| BLE 통신 | 100% |
| Firebase Realtime Database | 100% |

## 주요 경험

- Android 앱 전반을 직접 구현하며 기기 연동형 서비스 흐름 구축
- BLE 통신으로 운동 기기 데이터 수집 및 상태 처리
- SQLite를 활용한 로컬 데이터 저장 구조 구현
- Firebase Realtime Database와 연동한 수집 데이터 동기화 구조 구현
