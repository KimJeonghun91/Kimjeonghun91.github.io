# 아두이노 IOT 냉장고

카메라와 머신러닝으로 식품 상태를 분석하는 IoT 프로젝트 | 2020.06

## 프로젝트 개요

아두이노와 카메라 모듈로 냉장고 내부를 촬영하고, 머신러닝 분석을 통해 식품 상태를 판단한 뒤 앱으로 알림을 전달하는 프로젝트입니다.

하드웨어, 분석 로직, 모바일 알림 흐름이 연결된 IoT 서비스의 end-to-end 구조를 구현했습니다.

## 🛠 기술 스택

| 분야 | 기술 |
|------|------|
| 하드웨어 | Arduino, Camera Module |
| AI / ML | Machine Learning |
| 모바일 | Android |

## 주요 경험

- 아두이노와 카메라 모듈을 활용해 냉장고 내부 데이터를 수집하는 구조 구현
- 촬영 이미지를 머신러닝으로 분석해 식품 상태를 판단하는 흐름 구성
- 분석 결과를 모바일 앱 알림으로 연결해 사용자 액션까지 이어지는 구조 구현
