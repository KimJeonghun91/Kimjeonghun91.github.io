# 바라봄 (2021)

반려동물 건강 기록 서비스의 첫 운영 버전 | 2021.06

## 프로젝트 개요

바라봄은 반려동물의 건강 기록을 더 쉽고 꾸준하게 남길 수 있도록 만든 서비스입니다.

모바일 앱뿐 아니라 웹, 백엔드, 인프라까지 직접 구성하며 서비스 운영의 전체 흐름을 설계한 첫 프로젝트입니다.

## 🛠 기술 스택

| 분야 | 기술 |
|------|------|
| 모바일 | React Native |
| 백엔드 | Node.js (Express) |
| 웹 | Next.js |
| 인프라 | AWS |

## 주요 경험

- React Native 기반 모바일 앱을 개발하고 실제 서비스 운영 기반 마련
- Node.js(Express), MySQL, AWS로 백엔드와 운영 환경 구성
- Next.js 기반 웹 페이지까지 함께 운영하며 프론트엔드와 백엔드를 아우르는 구조 설계
- 이후 V26 재설계의 기반이 된 초기 제품 구조와 운영 흐름 구축
