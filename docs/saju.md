# 운세 상담 앱

채팅과 영상·음성 통화를 포함한 상담 앱 | 2018.06

## 프로젝트 개요

유저와 상담사가 채팅, 영상 통화, 음성 통화로 상담할 수 있는 서비스를 개발했습니다.

실시간 통신 기능뿐 아니라 일일운세, 무료사주, 인앱 결제까지 함께 다루며 앱 전반을 담당했습니다.

## 🛠 기술 스택

| 항목 | 기여도 |
|------|--------|
| React Native | 100% |
| In-App Purchase | 100% |
| WebRTC / App | 100% |
| WebRTC / Signal | 90% |

## 주요 경험

- React Native 앱 구조와 주요 기능 전반을 설계·구현
- 영상·음성 통화 기능을 WebRTC 기반으로 앱 레이어에 연결
- 인앱 결제를 포함한 유료 기능 흐름 구현
- 실시간 상담, 콘텐츠형 부가 기능, 결제가 함께 동작하는 서비스 구조 구축
